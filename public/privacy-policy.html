Privacy Policy

Last Updated: June 10, 2026

Introduction

Class Fund Manager is a student fund management platform used to collect and manage class contributions for events, programs, and activities. We are committed to protecting the privacy of our students and contributors.

This project is open source. While the source code is publicly available, user data, contribution records, authentication data, and payment information are not publicly exposed.

Information We Collect

User Information:

  • Name
  • Email Address
  • Roll Number
  • Student Information

Payment Information:

  • Razorpay Payment IDs
  • Transaction Amounts
  • Contribution Records

Payment card information is processed securely by Razorpay and is not stored on our servers.

How Information Is Used

We use the collected information for the following purposes:

  • Student authentication
  • Contribution tracking
  • Financial transparency
  • Program management
  • Communication regarding contributions

Data Storage & Security

Data is stored securely using Firebase. Access is strictly restricted to authorized administrators (Class Representatives). HTTPS is used for secure transmission of all data across the platform.

Third-Party Services

We use the following third-party services to operate the class fund platform:

  • Razorpay: For secure payment processing.
  • Firebase: For secure database hosting and authentication.
  • Google Authentication: For optional, secure sign-in mechanisms.

User Rights

Students and contributors may request:

  • Data correction
  • Account deletion
  • A copy of their contribution history

Contact Information

For any privacy-related questions, please visit our Contact page.
